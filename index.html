What is this?

Ninja Warrior is an exciting and funny platformer game made by 2 students from Development and Design of Videogames degree, in CITM, UPC.

Instructions to play:

Click on "Download" link to download the zip. Decompress it and open the folder whose name is "Ninja Warrior 2.0". Click on .exe file and enjoy!

Core subsystem:

Our game structure is divided into modules, when j1Module is the main module that shares awake, preupdate, update, postupdate and cleanup functions of all other modules. App is the module that calls the previous functions to be used by a certain module. One special mention is that we have 2 modules called j1EntityManager and j1GUI that manages all entity and UI submodules.

How to play:

D: Walk forward.
A: Walk backward.
SPACE: Jump.
D+LSHIFT: Run forward.
A+LSHIFT: Run backward.
D+LSHIFT+SPACE: Jump while running forward.
A+LSHIFT+SPACE: Jump while running backward.

Double jump!

SPACE+SPACE: Simple double jump.
D+SPACE+SPACE: Double jump while walk forward.
A+SPACE+SPACE: Double jump while walk backward.
D+LSHIFT+SPACE+SPACE: Double jump while running forward.
A+LSHIFT+SPACE+SPACE: Double jump while running backward.

Debug controls

F1: Change to map 1.
F2: Change to map 2.
F3: Start from the beginning of the current level.
F5: Save current state (player and enemies).
F6: Load current state (player and enemies).
F8: Debug UI.
F9: View colliders and logic.
F10: God mode.
F11: Framerate cap ON/OFF.
ESC (when user is in-game): Open pause menu.
